🔗 ربط الموقع بـ GitHub وتفعيل الاستضافة المجانية

📦 ما هو GitHub Pages وكيف يعمل؟

GitHub Pages هي خدمة استضافة مجانية من GitHub تتيح لك رفع موقع HTML/CSS/JS وجعله متاحاً للعالم على رابط مثل:

yourusername.github.io/your-repo-name
✅ مجاني تماماً لا حدود للزيارات، SSL مجاناً، CDN عالمي سريع، يدعم الدومين المخصص.

🚀 خطوات الربط — من الصفر للنشر (أول مرة)

1
إنشاء حساب GitHub

اذهب إلى github.com وأنشئ حساباً مجانياً. استخدم اسم بسيط مثل SmartTechEG — سيكون جزءاً من روابط مواقعك.

2
إنشاء Repository جديد

اضغط + → New Repository. سمّه باسم موقعك مثل smart-tech-website. اختر Public. لا تضيف README.

3
رفع ملفات الموقع

اضغط Add file → Upload files ثم اسحب الملفات: index.html, guide-complete.html, manifest.json, sw.js. اضغط Commit changes.

4
تفعيل GitHub Pages

في Repository اضغط Settings → Pages. في "Source" اختر Deploy from a branch. في Branch اختر main ثم /(root). اضغط Save.

5
موقعك الآن حي! 🎉

انتظر 1-3 دقائق ثم اذهب إلى: yourusername.github.io/repo-name — ستجد موقعك يعمل!

💡 نصيحة: ستجد الرابط الكامل في Settings → Pages تلقائياً بعد التفعيل.

🔄 تحديث الموقع بعد التعديل

1
الطريقة السهلة (مباشرة من GitHub)

افتح الملف في GitHub → اضغط أيقونة القلم ✏️ → عدّل → اضغط Commit changes. سيتحدث الموقع خلال دقيقة واحدة.

2
الطريقة الاحترافية (Git من الكمبيوتر)

حمّل Git ثم نفّذ:

git clone https://github.com/yourusername/repo-name
# عدّل الملفات على جهازك
git add .
git commit -m "تحديث الأسعار والعروض"
git push origin main

🎛️ كيف تتحكم في الموقع — لوحة الإدارة السرية

🔐 الدخول للوحة الإدارة

لوحة الإدارة مخفية ومحمية بكلمة مرور. للوصول إليها:

🖱️ الطريقة 1: النقر على الشعار انقر على شعار ST في أعلى الصفحة 5 مرات متتالية بسرعة. ستظهر نافذة إدخال كلمة المرور.
⌨️ الطريقة 2: رابط مباشر أضف #admin في نهاية رابط الموقع: yoursite.com#admin ثم اضغط Enter.
🔑 كلمة المرور الافتراضية: smart2025 — يمكنك تغييرها من داخل لوحة الإدارة.

⚙️ ما يمكنك التحكم فيه من لوحة الإدارة

📢 شريط الإعلاناتكتابة نص إعلان أو عرض مؤقت وإظهاره أو إخفاؤه بزر واحد. مثال: "عرض خاص — خصم 20% لفترة محدودة"
💰 تعديل الأسعارتغيير سعر الباقة الأساسية وأسعار الباقات الأخرى مباشرة دون تعديل الكود.
🏷️ شارة العرضتفعيل/إيقاف الشارة العائمة "عرض محدود" وتحديد نصها ونسبة الخصم.
📱 رقم الواتسابتغيير رقم الواتساب الظاهر في كل أزرار الموقع دفعة واحدة.
⏱️ مؤقت العد التنازليإضافة عداد تنازلي للعروض المحدودة لزيادة الإلحاح وتحفيز العملاء.
🎨 لون الشريطتغيير لون شريط الإعلانات بين الذهبي، الأخضر، والأحمر حسب نوع الإعلان.
💾 الحفظ التلقائي: جميع الإعدادات تُحفظ في localStorage المتصفح — تبقى حتى بعد إغلاق المتصفح.

⚠️ سياسة الدومين الرسمية لـ Smart Tech

⚠️ هام جداً: إخلاء مسؤولية الدومين
Smart Tech تقوم بتصميم وتطوير الموقع فقط. شراء ومتابعة الدومين (اسم الموقع مثل: yourname.com) هو مسؤولية العميل الكاملة. لا تتحمل Smart Tech أي مسؤولية عن انتهاء صلاحية الدومين أو فقدانه.

يشمل ذلك:

  • شراء الدومين من مزود موثوق (Namecheap، GoDaddy، Name.com، إلخ)
  • تجديد الدومين سنوياً (يبدأ من 10-15 دولار/سنة)
  • ربط الدومين بالاستضافة (DNS Settings) — Smart Tech تساعد في هذه الخطوة بمقابل رمزي
  • حفظ بيانات الدخول لحساب الدومين

🔍 تحسين SEO للظهور في Google

✅ ما تم تطبيقه بالفعل في الموقع

  • ✅ Meta Tags كاملة — عنوان، وصف، keywords، author
  • ✅ Open Graph — مشاركة جميلة على Facebook وWhatsApp
  • ✅ Schema.org JSON-LD — بيانات منظمة لـ Google
  • ✅ Canonical URL — تجنب المحتوى المكرر
  • ✅ Semantic HTML — section، nav، footer، h1-h3 مرتبة
  • ✅ Mobile Responsive — يعمل على جميع الشاشات
  • ✅ PWA — يمكن تثبيته كتطبيق
  • ✅ سرعة تحميل عالية — HTML/CSS/JS خالص
  • ✅ Arabic RTL — دعم كامل للعربية
  • ✅ Geo Tags — يستهدف مصر جغرافياً

كم عدد العملاء الذين يمكنني استضافتهم مجاناً؟

GitHub يتيح مواقع غير محدودة مجاناً ما دامت المشاريع Public. يمكنك إنشاء مئات الـ repositories.

هل يمكن للعميل تعديل موقعه بنفسه؟

نعم، إذا أعطيته صلاحية Collaborator على الـ Repository. أو يطلب منك التعديل وتفرض رسوم خدمة.

ماذا لو أراد العميل موقعاً بدومين .com؟

يشتري الدومين (مسؤوليته) ثم تربطه أنت بـ GitHub Pages عبر DNS Settings — خطوة بسيطة تأخذ 5 دقائق.

هل الموقع يعمل بدون إنترنت؟

نعم! بفضل Service Worker المدمج، بعد أول زيارة يُخزَّن الموقع ويعمل جزئياً offline.

كيف أضيف قاعدة بيانات للموقع مجاناً؟

Firebase Firestore (مجاني حتى 1GB) أو Supabase (PostgreSQL مجاني) أو MongoDB Atlas (512MB مجاناً).
